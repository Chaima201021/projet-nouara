Introduction Général

Cette ressource complète a été créée pour vous fournir toutes les informations nécessaires pour tirer le meilleur parti de notre plateforme innovante et de ses services. Que vous soyez un nouvel utilisateur à la recherche de conseils ou un utilisateur expérimenté souhaitant approfondir vos connaissances, ce Wiki est là pour vous accompagner à chaque étape.
L'objectif de ce guide est de vous fournir toutes les informations dont vous avez besoin pour commencer à utiliser Nouara.

C'est quoi NOUARA

Noura, notre plateforme de provisioning dédiée à nos partenaires. Avec NOUARA, explorez votre éligibilité, supervisez vos ventes, effectuez des commandes en toute simplicité, suivez vos opérations, créez des devis personnalisés, générez vos factures en quelques clics, et planifiez vos activités en toute efficacité.

Info Si vous avez besoin de plus d'informations, veuillez visiter le site NOUARA.

Se connecter

Si vous avez déjà un compte, vous pouvez vous connecter au portail Nouara depuis un PC ou un ordinateur portable à l'adresse https://nouara.fr avec votre adresse électronique et votre mot de passe.

Créez votre première TRUNK SIP

Avec Nouara, vous pouvez gérer votre premier trunk en toute liberté grâce à une interface conviviale et intuitive. Vous avez le contrôle total pour créer vos propres trunks SIP de manière instantanée. Alors que vous pourriez normalement attendre plusieurs jours pour voir une nouvelle adresse IP ajoutée, Nouara vous permet de le faire en un instant.
Voici comment procéder :
Accédez à la gestion des trunks : Une fois connecté, recherchez la section de gestion des trunks.

Créez un nouveau trunk SIP : Dans la section de gestion des trunks, vous devriez trouver un bouton ou une option pour créer un nouveau trunk SIP. Cliquez dessus pour commencer le processus de création.

Configurez les paramètres du trunk :
Lors de la création du trunk SIP, vous serez invité à fournir les détails nécessaires tels que le nom du trunk, les informations d'authentification et autres paramètres requis.



1.Générez un mot de passe sécurisé : Dans les paramètres du client, vous trouverez une option pour générer un mot de passe sécurisé. Utilisez cette fonctionnalité pour créer un mot de passe fort et protéger le compte du client contre les accès non autorisés.
2. Configurez les CPS (appels par seconde) : Selon les besoins du client, configurez les CPS dans les paramètres du compte. Cette valeur détermine le nombre maximum d'appels sortants par seconde autorisés pour le client.
3. Définissez le nombre total de canaux : Spécifiez le nombre total de canaux disponibles pour le client. Cela détermine le nombre maximum d'appels simultanés que le client peut passer.
4. Indiquez le nombre de canaux de réception : Configurez le nombre de canaux de réception autorisés pour le client. Cela détermine le nombre maximum d'appels entrants simultanés qu'il peut recevoir.
5. Choisissez le mode CLI (transparence ou forcé) : Sélectionnez le mode CLI souhaité pour le client. Vous pouvez choisir entre le mode de transparence, où le numéro de l'appelant est affiché tel qu'il est, ou le mode forcé, où vous pouvez spécifier un numéro d'affichage.
6. Sélectionnez la méthode de facturation : Déterminez la méthode de facturation préférée pour le client, qu'il s'agisse du prépayé, du post-payé ou de l'illimité. Cela détermine comment les frais d'utilisation sont facturés et payés.
7. Configurez le solde du compte et les seuils d'alerte : Spécifiez le solde initial du compte client et définissez les seuils d'alerte pour le solde. Chaque fois que le solde atteint un seuil spécifié, un e-mail automatique est envoyé pour informer le client de son solde et éviter les interruptions de service.

Si vous souhaitez ajouter une couche de contrôle supplémentaire à votre trafic sortant, Nouara propose des fonctionnalités pour définir le nombre maximum de tentatives par jour et par mois, ainsi qu'une planification horaire.
En utilisant ces fonctionnalités de contrôle supplémentaires, vous pouvez mieux gérer votre trafic sortant et ajuster les paramètres en fonction de vos besoins spécifiques. Assurez-vous de sauvegarder les modifications et de tester les configurations pour vous assurer qu'elles correspondent à vos attentes

Suivi

Une fois que le trunk a été créé, vous aurez la possibilité d'effectuer un suivi en temps réel de l'état du trunk, ainsi que de l'appareil auquel le tunk a été connecté.

Grille tarifaire

Avant de passer le premier appel, il est essentiel de définir la grille tarifaire pour votre service.
La grille tarifaire détermine les tarifs appliqués aux différents types d'appels et aux destinations spécifiques. Voici comment procéder pour définir une grille tarifaire :
1. Accédez à la grille tarifaire : Dans votre compte Nouara, recherchez la barre latérale ou le menu de navigation où se trouve l'option de la grille tarifaire. Cliquez dessus pour accéder à la gestion de la grille tarifaire.
2. Exportez votre exemplaire : Pour commencer, vous pouvez exporter un exemplaire de la grille tarifaire existante. Cette étape est facultative mais peut être utile pour avoir une base de départ ou pour consulter les tarifs actuels.
3. Téléchargez le fichier Excel : Lorsque vous exportez la grille tarifaire, un fichier Excel sera téléchargé sur votre ordinateur. Ce fichier contiendra les informations de tarification actuelles pour les différentes destinations.
4. Modifiez la grille tarifaire : Ouvrez le fichier Excel téléchargé et modifiez la grille tarifaire en fonction de vos besoins. Vous pouvez ajouter, supprimer ou modifier les tarifs pour les différentes destinations. Assurez-vous de respecter le format et les instructions spécifiques du fichier Excel.
5. Déposez et importez votre grille : Une fois que vous avez modifié la grille tarifaire dans le fichier Excel, retournez à la gestion de la grille tarifaire dans votre compte Nouara. Vous devriez trouver une option pour déposer et importer votre fichier Excel modifié. Suivez les instructions pour téléverser votre fichier.
6. Vérifiez et confirmez l'importation : Après avoir téléversé votre fichier Excel, Nouara vérifiera et importera les nouvelles informations de tarification. Vérifiez attentivement les détails importés et assurez-vous que la grille tarifaire correspond à vos attentes.

Si vous avez déjà défini votre grille tarifaire, vous pouvez l'affecter directement aux trunks de vos clients dans Nouara. Voici comment procéder :
Accédez à la barre latérale : Dans votre compte Nouara, recherchez la barre latérale où se trouve l'option pour affecter une grille tarifaire à un trunk. Cliquez dessus pour accéder à cette fonctionnalité.

1. Cliquez sur "Ajouter" : Une fois dans la section d'affectation de la grille tarifaire, recherchez le bouton ou l'option "Ajouter". Cliquez dessus pour commencer le processus d'affectation.
2. Choisissez le trunk : Dans la fenêtre d'affectation, vous devriez avoir une liste des trunks disponibles. Sélectionnez le trunk spécifique auquel vous souhaitez attribuer une grille tarifaire.
3. Choisissez la grille correspondante : Dans la même fenêtre, vous verrez une liste déroulante des grilles tarifaires disponibles. Choisissez la grille tarifaire qui correspond aux tarifs que vous souhaitez appliquer à ce trunk spécifique.
4. Enregistrez les modifications : Après avoir sélectionné la grille tarifaire appropriée pour le trunk, enregistrez les modifications apportées. Les tarifs de la grille tarifaire seront maintenant affectés à ce trunk.
